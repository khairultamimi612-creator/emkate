🏠 EMKATE Property
Website properti modern berbasis HTML, CSS, dan JavaScript untuk menampilkan daftar rumah dengan fitur interaktif seperti pencarian, filter, dan kontak langsung via WhatsApp.

🌐 Live Preview
👉 https://khairultamimi612-creator.github.io/emkate

✨ Fitur Utama
* 🔍 **Search Property** — Cari rumah secara cepat
* 🎯 **Filter Properti** — Berdasarkan tipe & harga
* 🏠 **Detail Rumah (Modal Popup)**
* 💬 **Chat WhatsApp Otomatis**
* 📱 **Responsive Design**
* 🎨 **UI Modern & Animasi**

## 📂 Struktur Project

1. Download / clone repository ini
2. Buka file `index.html`
3. Website langsung berjalan di browser
